# Secure File Sharing Platform

## Features
- Complete Authentication System
- Upload and securely store files
- Generate expiring and optionally one-time use share links
- Advanced Access Control: Require guests to request access via email. Owners can approve or deny.
- Modern responsive frontend without any frameworks.

## Setup & Run
1. Ensure Flask is installed:
   ```bash
   pip install Flask werkzeug
   ```
2. Run the application from the project root:
   ```bash
   python backend/app.py
   ```
3. Access at `http://127.0.0.1:5000`
